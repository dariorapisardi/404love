# Contributing to 404love

Thanks for helping improve 404love. Please keep changes focused, readable, and reusable.

## Quick start (npm)

- `npm install`
- `npm run build`
- `npm run test`

## Developing the Next.js package

- `npm --prefix packages/next run build`
- `npm --prefix examples/basic run dev`
- `npm --workspace @404love/next run test:example`

## Changesets

If you change the `@404love/next` package, add a Changeset:

- `npm run changeset`

## Issues and labels

- Star if useful. Open an issue for missing integrations, rescue sources, or accessibility gaps.
- Look for the `good first issue` label for starter tasks.
